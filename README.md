# SMNA2026 Assignment 2 — Virality Prediction on Reddit

## Research Question
Can the early network structure and sentiment of a Reddit post, 
measured within the first two hours of posting, predict whether 
it will go viral — defined relative to each subreddit's baseline 
engagement?
